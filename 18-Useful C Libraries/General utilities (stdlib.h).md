## General utilities (stdlib.h)
**stdlib.h** is a standard C library that provides **general-purpose utilities**

**Think of it as:** “tools for memory, conversion, random numbers, and program control”

### Main categories inside <stdlib.h>
### 1. Memory management
Used for **dynamic memory allocation**

```c
void *malloc(size_t size);
void *calloc(size_t n, size_t size);
void *realloc(void *ptr, size_t size);
void free(void *ptr);
```

#### Example
```c
int *arr = malloc(5 * sizeof(int));
free(arr);
```

### 2. Conversion functions
Convert strings → numbers

```c
int atoi(const char *str);
double atof(const char *str);
long strtol(const char *str, char **endptr, int base);
```

#### Example:
```c
int x = atoi("123");  // x = 123
```

#### 2.1 atoi() — string → int
```c
int atoi(const char *str);
```

#### What it does
Converts a string to an int

```c
int x = atoi("123");   // x = 123
```

#### Problems
- No error detection
- If invalid → returns 0 (ambiguous)

```c
atoi("abc");  // returns 0 (but was it really 0?)
```

#### 2.2 atof() — string → double
```c
double atof(const char *str);
```

#### Example
```c
double x = atof("3.14");  // x = 3.14
```

#### Same problem
- No error checking
- Unsafe for real programs

#### 2.3 atol() — string → long
```c
long atol(const char *str);
```

#### Example
```c
long x = atol("1000000");
```

#### Same issue
- No validation
- Cannot detect errors

#### 2.4 strtol() — string → long (SAFE)
```c
long strtol(const char *str, char **endptr, int base);
```

#### What makes it powerful?
- Detects errors
- Knows where conversion stopped
- Supports different bases (2–36)

#### Example
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    char *end;
    long x = strtol("123abc", &end, 10);

    printf("Value: %ld\n", x);     // 123
    printf("Rest: %s\n", end);     // "abc"
}
```

#### Key idea
- **Input:** "123abc", **result:** 123
- **Input:** endptr ->, **result:** "abc"

You can detect invalid input

#### 2.5 strtod() — string → double (SAFE)
```c
double strtod(const char *str, char **endptr);
```

#### Example
```c
char *end;
double x = strtod("3.14xyz", &end);

printf("%f\n", x);   // 3.14
printf("%s\n", end); // "xyz"
```

#### Same advantages as strtol
- Detects errors
- Partial parsing
- More control

### 3. Random numbers
```c
int rand(void);
void srand(unsigned int seed);
```

#### Example
```c
srand(10);
int r = rand();
```

#### 3.1 rand()
```c
int rand(void);
```

Returns a pseudo-random integer

A **pseudo-random integer** is a number generated by a deterministic algorithm (a Pseudorandom Number Generator or PRNG) that approximates the properties of random numbers. 
While these numbers appear random, they are calculated from an initial value called a seed and will repeat in the same sequence if the same seed is used

#### Key idea
It’s not truly random — it follows a sequence

#### Example
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    printf("%d\n", rand());
    printf("%d\n", rand());
    printf("%d\n", rand());
    return 0;
}
```

#### Output (example):
```text
1804289383
846930886
1681692777
```

#### Important problem

If you run the program again → you get the same numbers

Why? Because of the seed

#### 3.2 srand()
```c
void srand(unsigned int seed);
```

Sets the **starting point (seed)** of the random sequence

#### Key idea
Same seed → same random numbers

#### Example
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    srand(1);  // fixed seed

    printf("%d\n", rand());
    printf("%d\n", rand());

    return 0;
}
```

Always prints the same values

#### Real use (important)
Use **time as seed:**

```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));  // different every run

    printf("%d\n", rand());
    return 0;
}
```

#### Why this works
- time(NULL) changes every second
- So seed is different → sequence changes

#### Getting numbers in a range
#### Example: 0 to 9
```c
int r = rand() % 10;
```

#### Example: 1 to 100
```c
int r = rand() % 100 + 1;
```

#### Full example
```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));

    for (int i = 0; i < 5; i++) {
        printf("%d\n", rand() % 50 + 1);
    }

    return 0;
}
```

### 4. Program control
```c
void exit(int status);
void abort(void);
```

#### Example
```c
exit(0);   // normal termination
```

#### 4.1 exit()
Terminates the program normally

```c
void exit(int status);
```

#### What happens internally
When you call exit():
1. Calls functions registered with atexit()
2. Flushes buffers (printf, files)
3. Closes files
4. Returns control to OS

#### Example
```c
#include <stdlib.h>
#include <stdio.h>

int main() {
    printf("Before exit\n");
    exit(0);
    printf("After exit\n"); // never runs
}
```

#### Status codes
- 0 → success
- non-zero → error

#### 4.2 atexit()
Registers a function to run when exit() is called

```c
int atexit(void (*func)(void));
```

#### Key idea
When my program ends, run this function automatically

#### Example
```c
#include <stdio.h>
#include <stdlib.h>

void cleanup() {
    printf("Cleaning up...\n");
}

int main() {
    atexit(cleanup);
    printf("Program running\n");
    exit(0);
}
```

#### Output
```text
Program running
Cleaning up...
```

#### Important
- Functions are executed in reverse order (stack)
- Only runs if exit() is used (not abort())

#### 4.3 abort()
Terminates the program **abnormally (crash)**

```c
void abort(void);
```

#### Example
```c
#include <stdlib.h>

int main() {
    abort();  // program crashes immediately
}
```

#### Behavior
- Program stops immediately
- May generate a core dump
- Used when something is very wrong

#### 5. Searching and sorting
```c
void qsort(void *base, size_t n, size_t size, int (*cmp)(const void*, const void*));
void *bsearch(const void *key, const void *base, size_t n, size_t size, int (*cmp)(const void*, const void*));
```

#### 5.1 qsort()
Sorts an array **generically**

```c
void qsort(void *base, size_t n, size_t size,
           int (*cmp)(const void*, const void*));
```

#### Key idea
- Works with **any data type**
- You provide a **comparison function**

#### Example (sort integers)
```c
#include <stdio.h>
#include <stdlib.h>

int compare(const void *a, const void *b) {
    return (*(int*)a - *(int*)b);
}

int main() {
    int arr[] = {5, 2, 9, 1, 3};
    int n = 5;

    qsort(arr, n, sizeof(int), compare);

    for (int i = 0; i < n; i++)
        printf("%d ", arr[i]);

    return 0;
}
```
